# 型理論における分析的判断と総合的判断（Analytic and Synthetic Judgements in Type Theory）
**ペール・マルティン＝レーフ（Per Martin-Löf）** 

カントが分析的判断と総合的判断というよく知られた区別を導入したとき、それが全く新しいものではないことを彼は十分に承知していた。『プロレゴメナ』において、彼はロックへの明示的かつ非常に詳細な言及を行い、教条的な先達であるヴォルフやバウムガルテンがそれを疎かにしたことを非難している。また、ライプニッツによる理性の真理と事実の真理の区別を、彼が多かれ少なかれ当然のこととして知っていたと考えてよいだろう。もっとも、奇妙なことに、私の知る限り、彼はそれについて明示的な言及を一度も行っていないのだが。また、ヒュームによる観念の間の関係（relations of ideas）と事実に関する事柄（matters of fact）の区別も知られており、これはもちろん、ライプニッツの区別に言葉の上でも非常に近い。カント自身の用語法は、分析的判断対総合的判断というものであった。

カントの後、例えば、概念的命題と直観的命題（独 Begriffs- und Anschauungssätze）について語ったボルツァーノによっても、この区別が扱われている。ボルツァーノの場合、カントと比較すると状況は少し奇妙である。なぜなら、彼は概念的命題と直観的命題の区別だけでなく、分析的命題と総合的命題の区別も持っていたからである。しかしながら、彼は分析性と総合性の概念を完全に異なる方法で解釈した。そのため、ボルツァーノの分析的命題の概念は、今日我々が論理的に妥当な命題と呼ぶもの、すなわち、あらゆる可能な解釈の下で真となる命題であり、これはカントの概念とは全く異なるものである。一方、ボルツァーノの概念的命題と直観的命題の区別は、カントの分析的判断と総合的判断の区別に非常に近い。

最後に、ミルに言及したい。彼は一方で本質的命題と偶有的命題について語り（ここでの命題と判断の差異は重要ではない）、他方で言語的命題と実質的命題について語った。彼がこれら二組の概念の間で行った差異は、単に用語上のものに過ぎなかった。さて、ミルの最初の用語法は特に啓発的である。なぜなら、もし分析的判断と総合的判断の区別がそれほど重要であるならば、近代以前の論理学において、それに対応するものが何もなかったとしたら奇妙なことだからである。そして実際、ミルの最初の用語法は、それが何に対応するかを非常に明確に示していると私は思う。すなわち、アリストテレス論理学やスコラ論理学で非常に重要な役割を果たした本質的属性と偶有的属性の区別である。

さて、あるものに本質的属性が帰属せしめられる判断は分析的判断であり、一方、あるものに偶有的属性が帰属せしめられる判断、すなわち偶有的属性があるものに内在していることを述べる判断は、カントの用語法では総合的判断である。したがって、ミルの用語法、すなわち本質的命題対偶有的命題は、非常に適切に選ばれている。それは事柄の核心を直接的に示唆している。ちなみに、ミルの先へ進むと、本質的属性と偶有的属性の区別が再び現れる。それはウィトゲンシュタインによって『論理哲学論考』において、形式的性質と本来の性質（独 formale und eigentliche Eigenschaften）という区別の名の下に再発見されたようだが、問題となっているのは、本質的属性と偶有的属性という古くからの区別である。

分析的判断と総合的判断の区別を説明するために、カントは、説明的判断（独 Erläuterungsurteil）と拡張的判断（独 Erweiterungsurteil）という二つの用語を導入するという、非常に明快な手段を講じている。その考えとは、分析的、すなわち説明的判断とは、単なる概念分析によって自明となるもの、つまり、その自明性が概念分析のみに基づいているものである、というものである。その説明、あるいはほとんど逐語的にその説明は、カントによって与えられた。そして、分析的判断とはそれに含まれる用語の意味によって自明となるものであるという、我々全員が使用していると思われる現在の定式化、すなわち標準的な定式化を、私は確かに改善することはできない。

以上が、分析的、すなわち説明的判断に関する事柄であった。幸いなことに、それらに対する我々の現在の理解は、カント自身の理解と本質的に一致している。我々が困難を感じていると思われるのは、カントの総合的、すなわち拡張的判断という概念を適切に理解することである。カント的な考えでは、ある種の判断は、関わっている用語の意味だけでは自明ではなく、逆に、それを自分自身に対して自明なものとするためには、判断の内部に完全に含まれているものを越えて行かなければならない。もしそれが経験的判断、カントの用語で言えば後天的（a posteriori）判断であるならば、判断そのものを越えて向かう先は経験である。いわば外を見なければならない。一方、純粋に数学的な判断の場合、それを自明なものとするために判断自体に結合される必要があるのは、構成、すなわち数学的構成である。

カントは、「概念の構成を通じた数学的知識」（独 mathematische Erkenntnis durch die Konstruktion der Begriffe）という素晴らしい定式化を持っており、これこそがカントの総合的先天的（a priori）判断という概念を理解するための鍵である。したがって、総合的判断とは、自分自身を納得させるために判断自体を越えて行かなければならないものであり、純粋に数学的なケースでは、その「越えて行くこと」は、判断を自明なものとするために多かれ少なかれ独創的な構成を行わなければならないことを意味する。

型理論に入る前に、分析的判断と総合的判断の区別について、子供じみたほど単純な非数学的例をいくつか挙げてもよいだろう。そうすれば、これらの非数学的な例と、対応する型理論的な扱いとの間に大きな類似性があることを後で見ることになるだろう。例えば、これを見てほしい。

もし私が「雨が降っている」と言い、そして今、（実際は降っていないが）雨が降っていると仮定すると、この判断に含まれているものについての単なる概念分析をいくら行っても、雨が降っていることを教えてはくれない。もし雨が降っているという事実に自分自身を納得させたいのであれば、降りしきる雨に直接身をさらすか、あるいは何らかの間接的な証拠を提示する以外に方法はない。したがって、降りしきる雨、あるいはその間接的な証拠が「雨が降っている」という判断を自明なものとするのであり、それはこの判断が総合的であることを意味する。一方、言語的でない部分（降りしきる雨）と、適切な言語的部分（発話）からなる判断として、その複合体全体を捉えるならば、その判断は分析的なものである。

なぜなら、ご覧の通り、その判断には、自分自身を納得させるために必要なすべてが含まれているからである。もし、降りしきる雨の中にいて「雨が降っている」という言明に同意しない者がいるならば、その者の概念的理解に単純に誤りがある。彼は関わっている概念の一部を欠いているか、あるいはそれらを誤解しているといったことである。しかしながら、言語的なものと言語的でないものを同等に扱うこの方法は、カントにとっては異質なものであったに違いないことは直ちに認められなければならない。したがって、私は今、カントを越えることを述べた。

全く同様の分析が、誰かが「太陽が輝いている」と言う例や、あるいは次のような例にも当てはまる。「気温は＋２５℃である」という判断は、それ自体では確かに総合的なものである。自分自身を納得させるためには、そこに含まれる概念を分析するだけでは不十分である。しかし、その言語的表現が＋２５℃を示している温度計と一緒に捉えられるならば、その複合体全体はやはり明らかに分析的なものである。

例えば、この鉛筆の例を挙げよう。もし私が「この鉛筆は15cmの長さである」と言い、あなたがただその鉛筆を見るだけなら、その判断を自明なものとするには、関わっている用語の意味を思い出すだけでは不十分である。しかし、もし私が定規を持っていて、適切な方法でそれを定規に押し当て、あなたがその定規を読み取ることができるならば、「この鉛筆は15cmの長さである」という純粋に言語的な部分と組み合わされたその全体像は、確かに再び分析的なものとなる。すでにこれらの単純な例から、非常に重要なこと、すなわち、あらゆる総合的判断は分析的判断に基礎を置いており、総合的判断はいわば分析的判断、あるいは分析的連関のある一部分を抑制することによって得られるものである、ということがわかる。

これらすべてが、私の講演のタイトルにある型理論とどのような関係があるのだろうか。さて、我々が現在手にしている型理論は、単純型理論とは少し異なっており、その違いを正確に説明するが、それが採用している判断の形式を直接示すことによって、最も容易に記述される。まず第一に、それは $$a : \alpha$$ という形式の判断を行う。そのような判断は、$a$ は型 $\alpha$ の対象であると述べている。ここでカントなら型の代わりに範疇（category）、あるいは悟性の純粋概念と言ったであろう。この第一の形式の判断に加えて、同一性判断、すなわち $$a = b : \alpha$$ という形式の判断もある。この後者の形式の判断は、$a$ と $b$ は型 $\alpha$ の同一の対象であると述べている。ここで同一性は定義上の同一性と理解されるべきである。これらが、型理論で用いられる定言的判断（categorical judgement）の二つの基本形式である。

さて、システムの型はどのように生成されるのか。型の構造は、単純型理論の型構造の一般化である。チャーチによる単純型理論の定式化を思い出すならば、彼は命題の型として基本型 $\omicron$ を、個体の型として別の基本型 $\iota$ を持ち、それらの上に、もし $\alpha$ と $\beta$ が型であるならば、$\alpha$ から $\beta$ への関数の型（チャーチは $(\beta \alpha)$ と表記したが、私はシュッテの表記法 $(\alpha)\beta$ を好む）もまた型であるという節によって型構造を構築している。したがって、単純な型構造は以下の三つの規則によって生成される。
$$\omicron : \texttt{type}, \ \ \ \iota : \texttt{type}, \ \ \ \frac{\alpha : \texttt{type} \ \ \ \beta : \texttt{type}}{(\alpha)\beta : \texttt{type}}$$

直観主義型理論においては、これが一般化される。我々は依然として命題の型を持っているが（型理論では $\texttt{prop}$ と書かれる。これはチャーチの $\omicron$ と同じものである）、命題と集合の間のいわゆるカリー＝ハワード対応のために、それは集合の型と同一視され得る。したがって、直観主義型理論には、次の公理によって統治される集合の基本型が存在する。
$$\texttt{set} : \texttt{type}$$

これと命題の型、したがってチャーチの $\omicron$ も同一視される。
$$\texttt{prop} = \texttt{set} : \texttt{type}$$

しかし、個体の単一の基本型  を持つ代わりに、我々は次の規則を持つ。
$$\frac{A:\texttt{set}}{\texttt{elem}(A) : \texttt{type}}$$

これは、各集合 $A$ に対して、$A$ の要素が新しい基本型を形成することを述べている。論理的な読み方では、$A$ が命題として考えられるとき、$\texttt{elem}(A)$ は $A$ の証明の型としても機能する。
$$\texttt{proof}(A) = \texttt{elem}(A) : \texttt{type}$$

もしチャーチの単純型の表記法にできるだけ近づけたいのであれば、$\texttt{elem}(A)$ を単に $\iota(A)$ と書くことになるだろう。これは、我々が個体の単一の基本型（single ground type） $\iota$ を持っているだけではないことを示している。むしろ、各集合 $A$ に対して、その集合に属する個体の型 $\iota(A)$ を持っているのである。

さて、これらが基本型であり、チャーチの最初の二つの型形成規則を一般化することで得られたものである。そして、関数の型を形成するための規則の一般化も持っている。
$$\frac{\ \ \ \ \ \ \ \ \ \ \\ (x : \alpha) \atop \alpha : \texttt{type} \ \ \ \beta : \texttt{type}}{(x :\alpha)\beta : \texttt{type}}$$

これは、もし $\alpha$ と $\beta$ が型であり、そのうち $\beta$ が型 $\alpha$ の変数 $x$ に依存しているならば、依存関数型 $(x : \alpha)\beta$ もまた型になるということを述べている。すなわち、引数が型 $\alpha$ であり、引数 $x$ に対する値が型 $\beta$（これは $x$ に依存し得る）であるような関数の型である。これがチャーチの第三の型形成規則を一般化している理由は、$\beta$ が $x$ に依存しない特別な場合に、依存関数型の特殊ケースとしてチャーチの関数型が得られるからである。$\beta$ が $x$ に依存しない場合、$(\alpha)\beta$ は単に $(x : \alpha)\beta$ として定義され得る。
$$(\alpha)\beta = (x : \alpha)\beta : \texttt{type}$$

したがって、これらが、二つの基本形式の判断 $a : \alpha$ および $a = b : \alpha$ の右側に現れる型 $\alpha$ を生成する規則である。もし望むなら、私が今見てきたことを、カントの範疇（category）、すなわち悟性の純粋概念の形而上学的演繹（metaphysical deduction）の現代的類似物と見なしてもよいだろう。つまり、カントは、当時通用していた論理学の判断形式に着目し、使用されていた判断形式の完全なリストを作成することで、彼の範疇（category）に到達した。彼はこれをすべての悟性の純粋概念の発見への糸口と呼んだが、これは幸運なアイデアであった。

我々の場合、$a : \alpha$ および $a = b : \alpha$ に現れる $\alpha$ はまさに形式、あるいはむしろ二つ一組の判断形式である。したがって、我々は単に現在論理学で使用している判断形式がどのようなものであるかを反射的に確認すればよい。そして、これらの形式はすべて、関数型を形成する規則によって、あるものが集合であること、およびあるものが集合の要素であることを述べる二つの基本形式から生成され得ることが判明する。したがって、これは我々が現在型理論で使用している範疇（category）の網羅的なリストであり、範疇（category）、すなわち悟性の純粋概念を判断形式と同一視したカントの考えが、完全に損なわれることなく残っていることがわかる。

さて、どのような型 $\alpha$ を選ぼうとも、$a : \alpha$ および $a = b : \alpha$ という二つの判断形式はいずれも分析的である。なぜなら、これらの二つの形式のうちの一つの判断が自明であるとするならば、それはその中に現れる用語の意味のみによって自明だからである。では、総合的判断はどこから来るのか、と思われるかもしれない。それらは次のような方法で生じる。再び、$a$ が型 $\alpha$ の対象であることを表す第一の判断形式 $a : \alpha$ を考えよう。そのとき、我々はこの型 $\alpha$ の対象 $a$ が正確に何であるかについては気に留めず、単に型 $\alpha$ の対象の存在にのみ関心を持つことがよくある。そこで、例えば次の新しい判断形式を導入してもよいだろう。
$$\alpha \ \ \texttt{exists}$$

これは、「概念 $\alpha$ が存在性を持つ（the concept $\alpha$ has existence）」、あるいは「$\alpha$ がある（there exists an $\alpha$）」、あるいは、ボルツァーノや彼に続いたブレンターノが言ったように、単に「$\alpha$ はある（$\alpha$ is）」と読み替えることができる。これらはすべて、一つの同じ存在判断の形式の異なる読み方である。存在判断の意味は、次の規則から推し量ることができる。
$$\frac{a : \alpha}{\alpha \ \ \texttt{exists}}$$

より正確には、判断の形式を意味論的に説明するためには、その形式の判断を知るとはどういうことかを規定しなければならない。この場合、説明は、$\alpha \ \ \texttt{exists}$ であることを知るということは、$\alpha$ に属する対象を知ることである、というものである。

存在判断の形式は、特殊なケースとして、
$$A \ \ \texttt{is true}$$

という通常の判断形式を含んでいる。ここで $A$ は命題である。それを確認するために、我々の基本型の中に、任意の集合または命題 $A$ に対して、型 $\texttt{elem}(A)$、あるいは論理的な読み方では $\texttt{proof}(A)$ があったことを思い出そう。それは基本型であるから、その型、すなわち概念が存在性を持つという判断を形成することができる。
$$\texttt{proof}(A) \ \ \texttt{exists}$$

さて、この存在判断は、命題 $A$ の証明が存在することを述べている。したがって、真理性についての直観主義的な説明によれば、それはまさに命題 $A$ が真であることを述べている。なぜなら、真理性は直観主義的には、命題の証明、あるいは構成の存在として定義されるからである。したがって、通常の判断形式 $A \ \texttt{is true}$ は、確かに存在判断の形式の特殊なケースである。

これが分析的判断と総合的判断の区別とどのように関係しているのか。$a$ が型 $\alpha$ の対象であるということ、および $a$ と $b$ は型 $\alpha$ の同一の対象であるという二つの基本的な判断形式は、いずれも分析的であるとすでに述べた。総合的判断の形式とは、まさに私が今導入した存在判断の形式である。したがって、存在判断はカントの用語法では総合的である。その理由を知るために、その自明性が何に基づいているかを自問しなければならない。それは単に概念分析に基づいているのか、それともそれを自明なものとするために、判断の中に完全に含まれているものを越えて行かなければならないのか。さて、その自明性は構成に基づいている。我々は、概念 $\alpha$ に属する対象の構成を通じて、すなわちカントの用語法で言えば「概念の構成」（独 durch die Konstruktion des Begriffs）を通じて、$\alpha \ \texttt{exists}$ という存在判断に到達する。したがって、存在判断を自明なものとするためには、判断そのものに含まれているものを越えて、存在する物へと向かわなければならないことは明らかであり、ゆえにそれは総合的でなければならない。

カント自身が存在判断を総合的であると考えていたことは、実際には『純粋理性批判』に明示的に記されている。神の存在の存在論的証明の不可能性に関する節を見れば、「いかなる存在命題も総合的である」（独 daß ein jeder Existenzialsatz synthetisch sei）と彼が極めて明示的に述べているのを見つけるだろう。したがって、カントによれば、いかなる存在判断も総合的であった。そしてそれは、総合的判断の源泉についての彼の説明と正確に対応しているからである。私が始めた非数学的な例との類似性も、ここにはっきりと見て取ることができる。ここでもまた、総合的判断は、背後にある分析的判断のある一部分を抑制することによって得られている。したがって、あらゆる総合的判断は分析的判断に基礎を置いていることが再び確認された。その意味で、分析的判断の概念はもちろん、より基本的、あるいはより重要な概念である。

前述の分析性と総合性の概念の分析は、一つの重要な帰結をもたらす。皆さんがいくらか驚かれることを期待しているが、それは、通常の定式化における論理法則はすべて総合的であるという帰結である。カントによる「先天的（a priori）」という用語の使用を当然のこととして受け入れるならば、それらが先天的であることも明らかである。したがって、通常の定式化における論理法則は、総合的であるだけでなく、総合的先天的である。なぜなら、通常の定式化における論理法則はすべて、ある特定の形式の任意の命題は真であると述べており、肯定的な判断形式 $A \ \texttt{is true}$ は総合的判断の一形式だからである。

論理法則の単純な一例、例えば $A \supset (B \supset A \ \\& \ B)$ を取り上げよう。これは標準的な法則である。この法則が述べているのは、この形式のいかなる命題も真であるということであり、真理性についての直観主義的分析によれば、それはこの命題の証明が存在するということであって、これは存在形式の判断であり、したがって総合的である。では、標準的な方法でどのようにしてこの命題の真理性を納得させるのか。ある構成を行わなければならない。自然演繹の表記法を用いるならば、それは次のように読める。

![proofdiagram.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/62130/746f2662-fad2-434d-8a08-7331ad59c3c4.png)

したがって、この命題の真理性を納得させるためには、この構成を行わなければならない。型理論の表記法では、課題は、命題 $A \supset (B \supset A \ \\& \ B)$ の証明である構成 $c$ を見つけることである。記号的には $c : \texttt{proof}(A \supset (B \supset A \ \\& \ B)$ であり、その解は 
$$\begin{align}
&\supset I(A,B \supset A \ \\& \ B, (x) \supset I(B, A \ \\& \ B, (y) \ \\& \ I(A, B, x, y)))　\\\\
&: \texttt{proof}(A \supset (B \supset A \ \\& \ B))
\end{align}$$

である。しかし、ご覧の通り、これは型理論に特徴的な線形あるいは関数的な表記法で書かれた、上記の自然演繹風の証明に他ならない。本質的な点は、どのような論理法則を通常の定式化で取り上げようとも、その法則を自明なものとするためには、たとえそれがどれほど僅かであっても、独創性を必要とする小さな構成が行われなければならないということである。

ただし、述語論理はある図式的な命題が真であることを述べる論理法則のみから構成されているわけではない。それは項形成規則や、命題あるいは論理式を生成する論理式形成規則も含んでいる。したがって、述語論理は実際には次の三つの形式の判断を導出するための規則から構成されている。
$$\begin{align}
&t : \iota = \texttt{ind} \\\\
&A : \omicron = \texttt{prop} \\\\
&A \ \ \texttt{is true}
\end{align}$$

そして、これら最初の二つは第一の判断形式  の特殊なケースであり、それらは両方とも分析的である。総合的判断の形式であるのは、第三のものだけである。ケルナー教授が言われたように、カントはすべての数学的判断を総合的であるとは考えなかったが、興味深いものはすべて総合的であると考えた。そしてそれは全く正しい。興味深い数学的判断とは正確には存在的なものであり、数学的証明の困難さは、存在判断が存在性を帰属させる概念に属する対象を構成することにある。我々が現在開発しているコンピューター証明システムは、まさにこの点、すなわち与えられた概念に属する対象を構成するという困難なタスクにおいて機械的な支援を提供することが期待されている。

あとどのくらい時間があるだろうか。（10分です。）ああ、10分。では、分析的判断と総合的判断の区別が、伝統的な論理計算において我々が知っている不完全性現象や非決定性現象を適切に理解するために極めて重要であることを述べさせてほしい。状況はこうである。分析的判断の論理、すなわち二つの分析的形式の判断を導出するための論理は完全かつ決定可能である。一方、総合的判断の論理は、ゲーデルによって示されたように、不完全かつ非決定である。不完全性定理が総合的判断の論理に関わるものであることは明らかである。なぜなら、不完全性定理の証明の基礎となる決定的な判断は、$A \ \texttt{is true}$ という形式であり、これは存在判断の形式の特殊なケースであり、したがって総合的だからである。

分析的判断の論理が決定可能であるとはどういうことか。単に、二つの分析的形式、すなわち $a : \alpha$ および $a = b : \alpha$ のうちのいずれかの判断があるとき、その判断が形式的な規則によって導出可能であるか否かを決定することができるということであり、そのためのアルゴリズムがいわゆる型検査アルゴリズムである。また、それは次の意味で完全でもある。すなわち、もし特定の概念を含む分析的形式の判断があるならば、それを導出するために、まさにそれらの概念に関わる法則以外の法則は必要とされないということである。

では、不完全性現象や非決定性現象を生じさせるものは何なのか。それはもちろん、存在形式の判断 $\alpha \ \texttt{exists}$ において、型 $\alpha$ の対象 $a$ が抑制されているという事実である。分析的な $a : \alpha$ から総合的な $\alpha \ \texttt{exists}$ への移行において、決定可能性が失われる。$\alpha \ \texttt{exists}$ を証明するために、我々は型 $\alpha$ の対象 $a$ を探索しなければならず、その探索の手続きは必ずしも終了するとは限らない。特に、$\texttt{proof}(A)$、あるいは同じことだが $A \ \texttt{is true}$ という判断形式において、証明対象 $a : \texttt{proof}(A)$ が抑制されているという事実こそが、非決定性および不完全性の源泉である。つまり、再び命題 $A$ がある言語あるいは体系において定式化されているとして、$A \ \texttt{ is true}$ という形式の判断の、最も単純かつ最も重要なケースを考えてみよ。
そのとき、$A$ はその体系において証明不可能であるかもしれない。それは単に、その中で表現できる証明あるいは構成 $a : \texttt{proof}(A)$ が存在しないからという理由による。
他方で、元の体系に新しい概念を導入して何らかの拡張を行うならば（例えば算術から算術的解析や一般化された帰納的定義の理論などへ移行するように）、命題 $A$ が表現された元の体系では不可能であったとしても、拡張された体系において対象 $a : \texttt{proof}(A)$ を構成することができるようになることは、もちろん十分にあり得ることである。
ゲーデルが、例えば元の体系を第一階算術とし、第一階算術において $A$ の証明が存在しないような特定の第一階算術的命題 $A$ を構築することによって発見したのは、この現象であった。他方で、彼はまず非公式に、それにもかかわらずそのような証明が存在すること、すなわち $A$ は真であることを示し、それは例えば反映原理（reflection principle）を加えることによって得られる第一階算術の適切な拡張において、形式化することも可能である。したがって、分析的判断と総合的判断の区別が、不完全性現象および非決定性現象を適切に理解するために極めて重要であると私が述べたときに念頭に置いていたのは、このことであった。
